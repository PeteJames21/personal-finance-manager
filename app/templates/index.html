App's Features

Register

Once you create an account, Personal Finance Tracker creates an account and assigns the default profile of Personal. Profiles are the container that holds all of one's accounts in one place. In each profile, a user can create accounts which correspond to the containers that hold money e.g. Cash, Credit Card, MPesa etc.

Add Income/Expense

To add an income/expense, click on the add transaction button, toggle down and select either income, expense or transfer. For income, you need to enter the account debited, for expense an account credited is required while for transfer, bot fields are necessary. The Source/Destination documents where the money came from or the purpose of the expenditure.

Dashboard

The home page provides an overview of the user's transactions. The filter by date allows for filtered responses between certain periods of time. Te dashboard displays the total incomes, expenses and te net income. It allows displays a list of top 5 items under each.

About the Project

This project arose from a mutual desire to have a personalized application for tracking incomes and expenses. We are both freelancers, and our incomes tend to be unpredictable and outside the bounds of monthly or weekly income. There was a pressing need to keep track of our individual finances and be aware of incomes and expenditures. We carried the profile feature to increase the utility of the application as it can be used for non-personal uses such as managing corporate money. For our ALX SE project, we decided to develop this application - talk of killing two birds with one stone!
